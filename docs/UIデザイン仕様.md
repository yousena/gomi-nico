# ごみニコ UIデザイン仕様

最終更新: 2026-07-16（旧「進捗まとめ.md」のデザイン仕様部分を移設・更新）

---

## 技術構成

- 静的HTML + Vanilla JS（script.jsはES5互換、`var`/`function`宣言ベース）
- Tailwind CSS（Play CDN。本番ビルド移行は将来タスク）
- 志木市アプリ（shiki/index.html）は Tailwind + カスタムCSS変数（`--c-*`）併用
- データは `data_shiki.json`（地区別収集スケジュール・カテゴリ・品目DB・FAQ）

## デザイン方針（C案）

- **カード背景**: 白（`#fff`）+ 薄枠線（`1px solid rgba(0,0,0,0.07)`）で統一
- **カテゴリ色**: アイコンコンテナ背景（`iconBg`）のみに限定
- **ページ背景構造**: `html, body` = `#fff` / `<main>` = `#F7F6F3` / ヘッダー・フッター = 白
  - iOSバウンス時にヘッダー・フッターと背景が一致し、途切れない

## カラー

- ブランド: `#00A86B`
- カテゴリ: 可燃 `#E8512A` / 不燃 `#4B72D4` / 資源 `#00A86B` / 粗大 `#6878A0` / 危険 `#D96A00`

## SVGアイコン統合（アプリ側）

- 全SVGは透明背景（白rect削除済み）。`/icons/` に12種
- `TYPE_STYLE` に `img` プロパティ、`catIcon()` がSVG/Material Symbolを自動切替（20px以上でSVG）
- `selfBg: true`（kiken / yugai）: SVG内部に色付き矩形を持つため透明コンテナでフルサイズ表示

### アイコンコンテナ（アプリ側・4箇所）

| 表示箇所 | コンテナサイズ |
|---|---|
| 今日・明日のごみカード | 48px |
| ガイド（カテゴリ一覧行） | 44px |
| カテゴリ詳細ヘッダー | 52px |
| 品目詳細ヘッダー | 56px |

| パターン | 背景 | 画像サイズ |
|---|---|---|
| `selfBg: true`（危険/有害） | transparent | コンテナと同サイズ |
| SVGあり（通常） | `iconBg`（淡色） | コンテナ − 8px |
| Material Symbolのみ | `iconBg`（淡色） | 元サイズ |

※トップLP側は別ルール: 60×60px・角丸14px・背景タイルなしで直接表示

## 五十音リスト バッジ

- 28×28px 角丸正方形（`border-radius: 4px`）、アイコン単体（`iconBg`背景）

## カレンダーUI

| アイコン数 | サイズ | border-radius | gap |
|---|---|---|---|
| 1件 | 30×30px | 4px | — |
| 2件 | 20×20px 各 | 4px | 2px |
| 3〜4件 | 20×20px 各 | 4px | 2px（グリッド） |

- `.cal-cell` padding: `2px 0 2px` / 日付: 16px・weight 500
- 収集日背景色（has-col）は廃止（ホバーのみ薄グレー）
