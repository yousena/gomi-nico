## 📌 検索のひらがな・カタカナ・漢字表記ゆれ対応（2026-07-21）

- **背景**: 「ひらがな・カタカナ・漢字のどの表記でも検索・サジェストにヒットさせられないか、競合はできているか」との相談。競合最大手の「さんあ〜る」（全国の自治体が導入する分別アプリ）で「ふとん」（ひらがな）「布団」（漢字）どちらでも検索できることをWeb検索で確認し、業界標準機能として実装
- **カタカナ⇄ひらがな正規化を実装**: `script.js`に`toHiragana()`を新設。カタカナ(U+30A1-U+30F6)はひらがな(U+3041-U+3096)から固定オフセット(+0x60)のため、Unicode変換だけで表記ゆれを解消できる。`onSearch()`の一致判定に組み込み、「カン」で検索しても「かん」タグの品目が、逆も同様にヒットするように修正（ライブラリ不要・追加のページ重量なし）
- **漢字⇄かなは辞書化が困難なため、高頻度な品目に手動でひらがな読みタグを追加**（20品目・電池→でんち、自転車→じてんしゃ、蛍光灯→けいこうとう、缶→かん、雑誌→ざっし、毛布→もうふ、消火器→しょうかき、冷蔵庫/冷凍庫→れいぞうこ/れいとうこ、洗濯機→せんたくき、灯油→とうゆ、車椅子→くるまいす、金庫→きんこ、ドラム缶→どらむかん、農薬→のうやく、太陽光パネル→たいようこうパネル、石/砂/土→いし/すな/つち、屋根材/瓦→やねざい/かわら、画鋲/針→がびょう/はり）。自動ふりがな変換ライブラリ（kuromoji等）は誤読リスクとページ重量増があるため今回は見送り、正確性を優先し手動タグ付けを継続する方針

- **全品目への展開（同日）**: 「残り全品目にも読みタグを追加すべきか、SEO・UX・ページ速度で総合判断してほしい」との相談を受け検討。判断は以下の通り：

**「ごみの分別で、まちをニコッと。」** に確定。鳥貴族「焼鳥で世の中を明るくする」型の一行コピー。「世の中」ではなく「まち」にすることで、自治体単位で展開する事業の実態と一致させた。

- ただし「①アフィリエイトで使う楽天会員登録／②受取先の楽天銀行／③楽天カード」の**いずれかが法人名義だと、3,001円（楽天キャッシュ）以上の報酬受け取りに法人審査が必要**（楽天公式ヘルプ [000001114](https://affiliate.faq.rakuten.net/detail/000001114) より）。裏を返すと、①〜③をすべて宮澤さん個人名義に揃えれば審査不要で使える
