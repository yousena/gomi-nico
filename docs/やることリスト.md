- **判断軸として採用**: 開発・デザイン・広報で迷ったときは「これによってユーザーが『ニコッ』とするか」を唯一の意思決定基準にする（Geminiの提案を採用）
- **「住民・行政・地域企業/収集作業員・子どもの未来・環境の5つの笑顔」という展開案は不採用**: 列挙すると企業研修資料的になり「ダサい」との判断。上記の判断軸一行に集約し、公開コピーとしては使わない。縫い針を「きけん」表示にする等、個別の実装判断の中に思想として残ればよい、という位置づけ

### データ活用ロードマップ（Gemini案を採用・実行可能性で仕分け）

- **今すぐ着手可能**: 「迷子ごみログ」＝`search_no_result`（GA4、実装済み）の定期棚卸し。加熱式タバコ・新素材製品など現行データベース未対応品目の早期検知に使え、次の品目追加・記事テーマの優先順位づけにそのまま使える
- **戸田市・蕨市等の追加後**: 「自治体別・捨て方に迷うごみランキング」（比較には複数自治体の同時稼働が前提。志木市単独では成立しない）
- **半年〜1年分のログ蓄積後**: 「季節・生活イベント別の検索トラフィック変動」（転入期・大掃除期等。短期間のデータで語ると数字が誇張になるため、十分な期間を待つ）

### PR TIMES・広報戦略（Gemini案を採用、2点の留保つき）

- テーマ構成の型（季節トレンド型／事故防止・地域課題解決型／社会課題・環境型）は採用
- **留保①誇大な因果表現は避ける**: 「清掃車の火災事故を防ぐ！」等、証明できない因果関係を主張する見出しはDS.mdのボイス&トーン方針（誇大表現・未検証の数値NG）に抵触するため、「検索数の増加から見える関心の高まりを可視化」等、観測事実ベースの表現に言い換える
- **留保②統計的に意味のある量のデータが貯まるまでは送らない**: 現状のトラフィック規模では説得力のある数字にならない。テーマの型だけ保存しておき、実際の配信はユーザー数が積み上がってから
- **既存の3段階戦略と接続**: 「行政・地元企業への統計レポートのフィードバック」は、上記「第2段階: 協賛企業を味方にする」「第3段階: 自治体を味方にする（低姿勢な打診）」の材料としてそのまま使える。新しい概念ではなく、既存ロードマップの中に統計データという武器が一つ増えた、という位置づけ
- PR TIMESの出稿条件（無償の「スタートアップチャレンジ」は日本国内の株式会社・設立24ヶ月以内等が条件）は要確認。株式会社ユウセナが対象になるかは太平さんに確認が必要

### 個人名義での利用について（楽天公式ヘルプで確認済み）

- 楽天アフィリエイトは**個人利用なら審査なし**。楽天IDでログインするだけで即日から利用可能
- ただし「①アフィリエイトで使う楽天会員登録／②受取先の楽天銀行／③楽天カード」の**いずれかが法人名義だと、3,001円（楽天キャッシュ）以上の報酬受け取りに法人審査が必要**（楽天公式ヘルプ [000001114](https://affiliate.faq.rakuten.net/detail/000001114) より）。裏を返すと、①〜③をすべて宮澤さん個人名義に揃えれば審査不要で使える
- **検討すべき点**（会計・税務。判断は顧問税理士等に要確認）:
  - ごみニコの運営者情報ページには「株式会社ユウセナ」と明記済み。個人アカウントで受け取ると報酬は宮澤さん個人の口座に入るため、法人の売上として計上するには社内で資金の流れを整理する必要がある
  - 個人受け取りの場合、金額次第で宮澤さん個人の確定申告（雑所得等）が必要になる可能性がある
  - 規約上「同一サイトを法人と個人で使い分けてよいか」は明記が見つからなかったため、実施前に楽天アフィリエイト事務局へ確認するのが確実

### 代替ルート（ASP経由）

- **もしもアフィリエイト**: 会員登録の審査は緩やか（公序良俗違反等がなければ基本通過）。既にコンテンツのあるサイトが前提条件（ごみニコは条件を満たす）。承認されればAmazon・楽天どちらの商品リンクも同一アカウントで扱える
- **A8.net**: 入会審査自体はなし。ただし広告主（Amazon等）ごとに個別審査があり、そこでの通過率はもしもアフィリエイトの方がやや有利という情報が複数ある
- **Amazonアソシエイト直接**: サインアップから180日以内に3件以上の適格販売が必要、かつ最低10記事必要。トラフィックが少ない立ち上げ期には不向き
