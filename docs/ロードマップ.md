# ごみ分別ガイド サイト構築ロードマップ

最終更新: 2026年6月30日

---

## フェーズ1｜まず公開する
**目安: 1〜2日 ／ 優先度: 🔴 最優先**

- [ ] GitHub アカウントを作成（無料）
- [ ] index.html・script.js・data_shiki.json を GitHub にアップロード（ブラウザ操作のみ）
- [ ] Cloudflare アカウントを作成（無料）
- [ ] Cloudflare Pages と GitHub を連携（ボタン操作のみ）
- [ ] ドメインを取得（Cloudflare で購入が一番シンプル／年1,000〜2,000円）
  - 複数自治体に展開するなら市名を含まないドメインにする
  - 例: `gomi-nico.jp` / `bunbetsu-navi.jp` など
- [ ] Cloudflare Pages にドメインを設定（SSL は自動）

**完了したら:** `https://あなたのドメイン/?city=shiki` でサイトが公開される

---

## フェーズ2｜SEO基盤を整える
**目安: 半日 ／ 優先度: 🔴 最優先（フェーズ1と同時進行でOK）**

- [ ] `index.html` に以下を追加（Claude に依頼すれば作成可）
  - `<meta name="description">` — ページ説明文
  - OGP タグ（SNSシェア時のタイトル・画像）
  - JSON-LD 構造化データ（検索結果リッチ表示）
- [ ] **Google Search Console** に登録（Google にサイトの存在を通知）
- [ ] **sitemap.xml** を作成・Cloudflare Pages に設置・Search Console に送信
- [ ] **Google Analytics** を設置（アクセス数・流入キーワードの把握）

**完了したら:** 「志木市 ゴミ分別」などで検索結果に表示され始める

---

## フェーズ3｜AdSense・収益化
**目安: 申請〜審査1〜2週間 ／ 優先度: 🟡 早めに**

- [ ] Google AdSense に申請（フェーズ2完了後に申請すると審査が通りやすい）
- [ ] 審査通過後、広告コードを index.html に設置
- [ ] 楽天アフィリエイトのリンク・バナーを整備

**ポイント:** AdSense 審査はサイト公開済み＋コンテンツ充実が条件。フェーズ2を先に終わらせること。

---

## フェーズ4｜スプレッドシート連携
**目安: 1週間 ／ 優先度: 🟢 自治体が2つ目に増えたとき**

- [ ] Google スプレッドシートのテンプレートを作成（Claude に依頼）
  - シート構成: 基本情報 / ごみ一覧 / 収集スケジュール / 地区マスタ / FAQ
- [ ] Google Apps Script で自動化（Claude に依頼）
  - スプレッドシートの「公開」ボタン → JSON変換 → GitHub更新 → 自動デプロイ
- [ ] 自治体担当者向けマニュアルを作成

**完了したら:** 担当者がスプレッドシートを編集してボタンを押すだけでウェブ更新される

## フェーズ5｜コンテンツSEOと静的化
**目安: 継続的 ／ 優先度: 🟢 半年〜1年後**

### コンテンツSEO
- [ ] ブログ・記事ページを追加
  - 記事例:「志木市で粗大ごみを出す方法・申込手順」
  - 記事例:「引越し前に知っておきたい志木市のごみルール」
  - 記事例:「リチウム電池・モバイルバッテリーの捨て方（令和8年改定）」
- [ ] 志木市公式サイト・地域ポータルからのリンク獲得を目指す

### 静的化（Astro 移行）
- [ ] Astro フレームワークへ移行（Claude に依頼）
  - 現在: HTML読込 → JS実行 → JSON取得 → 描画（約1〜2秒）
  - 移行後: HTML読込 → 即表示（約0.1〜0.3秒）
- [ ] Astro でビルドし Cloudflare Pages にデプロイ
