## UIコンポーネント実装状況

### デザイン方針（C案）
- **カード背景**: 白（`#fff`）+ 薄枠線（`1px solid rgba(0,0,0,0.07)`）で統一
- **カテゴリ色**: アイコンコンテナ背景（`iconBg`）のみに限定
- **ページ背景構造**: `html, body` = `#fff` / `<main>` = `#F7F6F3` / ヘッダー・フッター = 白
  - iOSバウンス時にヘッダー・フッターと背景が一致し、途切れない

### SVGアイコン統合
- 全SVGの白背景（`<rect fill="white"/>`）を削除 → 透明背景
- `TYPE_STYLE` に `img` プロパティを追加し、`catIcon()` 関数でSVG/Material Symbolを自動切り替え（20px以上でSVG使用）
- `selfBg: true` フラグ（`kiken` / `yugai`）: SVG内部に色付き矩形を持つため透明コンテナでフルサイズ表示

### アイコンコンテナ（4箇所・3パターン分岐）

| 表示箇所 | コンテナサイズ |
|----------|--------------|
| 今日・明日のごみカード | 48px |
| ガイド（カテゴリ一覧行） | 44px |
| カテゴリ詳細ヘッダー | 52px |
| 品目詳細ヘッダー | 56px |

| パターン | 背景 | 画像サイズ |
|---------|------|-----------|
| `selfBg: true`（危険/有害） | `transparent` | コンテナと同サイズ |
| SVGあり（通常） | `iconBg`（淡色） | コンテナ − 8px |
| Material Symbolのみ | `iconBg`（淡色） | 元サイズ |

### 五十音リスト バッジ
- 28×28px 角丸正方形（`border-radius: 4px`）、アイコン単体のみ（`iconBg` 背景）

### カレンダーUI

| アイコン数 | サイズ | border-radius | gap |
|-----------|--------|--------------|-----|
| 1件 | 30×30px | 4px | — |
| 2件 | 20×20px 各 | 4px | 2px |
| 3〜4件 | 20×20px 各 | 4px | 2px（グリッド） |

その他設定:
- `.cal-cell` padding: `2px 0px 2px`
- 日付フォント: `font-size:16px; font-weight:500`
- 収集日背景色（`has-col`）: 削除（ホバーのみ薄グレー）

## メモ

- `script.js` はすべて Vanilla JS（ES5互換、`var` / `function` 宣言ベース）
- `data.json` に志木市全域の収集スケジュール（地区別）が格納済み
- 静的ページ（index / about / privacy / ads）は Tailwind CSS CDN を使用
- 志木アプリページ（shiki/index.html）は Tailwind + カスタム CSS 変数（`--c-*`）の組み合わせ
