## UIコンポーネント実装状況

### SVGアイコン統合
- 全SVGの白背景（`<rect fill="white"/>`）を削除 → 透明背景に変更
- `TYPE_STYLE` に `img` プロパティを追加し、`catIcon()` 関数でSVG/Material Symbolを自動切り替え（20px以上でSVG使用）

### アイコンコンテナ（4箇所）
| 表示箇所 | コンテナサイズ |
|----------|--------------|
| 今日・明日のごみカード | 48px |
| ガイド（カテゴリ一覧行） | 44px |
| カテゴリ詳細ヘッダー | 52px |
| 品目詳細ヘッダー | 56px |

**コンテナの背景ルール（3パターン）：**
- `selfBg: true`（危険/有害）→ 背景 `transparent`、画像フルサイズ（アイコン自身が色背景を持つため）
- SVGあり（通常）→ 背景 `#fff`（白）、画像は8px小さく配置
- Material Symbolのみ → 背景 `iconBg`（カテゴリの淡い色）

### 五十音リスト バッジ
- 旧: 横長ピル（アイコン＋カテゴリ名テキスト）
- 新: 28×28px 角丸正方形、アイコン単体のみ表示（`iconBg` 背景）

### カレンダーUI
- `.cal-cell` padding: `2px 0px 2px`
- 日付フォント: `font-size:16px; font-weight:500`
- 収集日背景色（`has-col`）: 削除（ホバーのみ薄グレー）
- アイコンが1種類の日: サイズ 32px で表示

## メモ

- `script.js` はすべて Vanilla JS（ES5互換、`var` / `function` 宣言ベース）
- `data.json` に志木市全域の収集スケジュール（地区別）が格納済み
- `TYPE_STYLE` の `selfBg: true` フラグ：`danger.svg`・`harmful.svg` はSVG内部に色付き矩形を持つため、白コンテナではなく透明コンテナで表示
