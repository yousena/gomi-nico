最終更新：2026-07-23

> 品目詳細シート用の個別品目イラストを、プロのイラストレーターに発注する際の叩き台。
> スタイル仕様の正典は `ごみニコDS.md` §2-4-2。本書は対象品目リストと発注実務のメモ。
> 対象範囲・スタイル方針は太平さんと合意済み（横浜市公式アプリ・志木市公式PDF「家庭ごみと資源の分け方・出し方」21940.pdf を参考）。

---

## 1. 目的

品目詳細シート（アプリでカテゴリ・品目をタップした時に開く画面）に、現状テキストのみで表示している品目に専用の色+線画イラストを追加し、視認性・親しみやすさを向上させる。自治体PDF・他社アプリで一般的な「色+線画」スタイルを、ごみニコのブランドカラーに合わせて取り入れる。

## 2. スタイル仕様（イラストレーターへの必須共有事項）

| 項目 | 仕様 |
|---|---|
| キャンバス | viewBox `0 0 80 80`（既存の`/icons/*.svg`と統一） |
| 描画方法 | **線のみ**で形を表現。塗りは白または透明。陰影・グラデーション・立体感（3D風）は入れない |
| 線の色 | 品目が属するカテゴリの色1色のみ（下表参照）。1イラスト内で複数色を混在させない |
| 線の太さ | 3〜4px（80×80キャンバス基準） |
| 線の端・角 | 丸め処理（round cap / round join）。角ばった鋭い線は避け、親しみやすい印象にする |
| 背景 | 透明（アプリ側のコンテナ・背景色に乗せて表示するため） |
| NG | 参考にした自治体PDF・他社アプリのイラストの直接トレース・複製（著作権上必須）。複雑な書き込み・リアルな質感表現・キャラクター的な擬人化 |

## 4. 発注・納品仕様

- 形式: SVGデータ一式（viewBox 0 0 80 80 統一）
- ファイル名: 英字の品目キー（例: `petbottle.svg`、`tv.svg`）。命名は実装時にgomi-nico側で指定する
- 色: 上記カテゴリ色で納品してもらい、実装側でCSS変数への差し替えを行う
- 参考画像として自治体PDF・他社アプリのスクリーンショットを渡すのは問題ないが、**「雰囲気の参考であり、直接のトレース・模写は不可」**と発注時に明記すること

## 5. 発注の進め方（推奨）

65点を一括発注すると、最初の数点でスタイルの方向性が合わなかった場合の手戻りコストが大きい。以下の2段階を推奨：

1. **スタイル確定フェーズ**: 5〜10点（例: ペットボトル・カン・紙パック・テレビ・自転車など、カテゴリを横断して代表的なもの）を先行発注し、線幅・丸み・色の実装結果を確認
2. **量産フェーズ**: スタイル確定後、残り全点を同じ絵師に同一スタイルで発注

## 6. 費用感（2026-0723時点の相場調査ベース）

- カラーの小カットイラスト相場: 1点5,000〜10,000円
- 65点想定で総額32.5万〜65万円程度。まとまった点数の一括発注はボリュームディスカウントが効くことが多く、実際の見積りは個別確認が必要
- 詳細は本チャットでの相場調査を参照（ココナラ・ランサーズ・日本イラストレーター協会等の情報）
